# Clarence CCNA Mastery

Structured, project-based CCNA mastery program designed to build real-world engineering skill and a verifiable portfolio.

## What you'll find here
Each section contains (as applicable):
- Packet Tracer topologies (`topology/`)
- Raw CLI configs (`configs/`)
- Verification output (`verification/`)
- Failure notes (misconfigs, symptoms, fixes)

## Operating rules
- No command without understanding (mode, purpose, subsystem/plane impact, failure/security implications).
- Sequential real-world configuration order.
- Verification before/after.
- Daily artifacts committed to GitHub.
